Class rio.Application


Extends rio.Attr.
Application is used to create new rio application classes. It provides functionality for dependency management, routing, history management and page management.
Defined in: application.js.

Creates an instance of rio.Application.

			requireCss: ["css_reset", "example"],

				"": "examplePage"

			attrAccessors: [],
			attrReaders: [],
